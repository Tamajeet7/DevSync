# DevSync

> AI-powered developer collaboration platform built for modern software teams.

## Overview

DevSync is a full-stack collaboration platform that brings project management, team communication, GitHub integration, and AI-assisted code reviews into one workspace.

---

## Features

- JWT Authentication
- Team Workspaces
- Project Management
- Kanban Boards
- Team Chat
- GitHub Integration
- AI Code Reviews
- Notifications

---

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

### Backend

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
